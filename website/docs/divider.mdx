# Divider

## import

```jsx
import { Divider, DividerTitle } from '@bone-ui/divider'
```

## Basic

```jsx live
<Divider></Divider>
```

## Vertical

```jsx live
<Box h-60 toCenter>
  <Divider orientation="vertical"></Divider>
</Box>
```

## With title

```jsx live
<Divider>
  <DividerTitle>INTRO</DividerTitle>
</Divider>
```

## With title vertical

```jsx live
<Box h-100 toCenter>
  <Divider orientation="vertical">
    <DividerTitle>INTRO</DividerTitle>
  </Divider>
</Box>
```

## Vertical Example

```jsx live
<Box toCenter spaceX2 h-20>
  <Box>Item1</Box>
  <Divider orientation="vertical"></Divider>
  <Box>Item2</Box>
  <Divider orientation="vertical"></Divider>
  <Box>Item3</Box>
</Box>
```

## Customize style

```jsx live
<Box spaceY4>
  <Divider bgRed500></Divider>
  <Divider h-2></Divider>
  <Divider h-4></Divider>
  <Divider h-8></Divider>
</Box>
```
