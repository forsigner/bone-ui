# CloseButton

## import

```jsx
import { CloseButton } from '@bone-ui/close-button'
```

## Basic

```jsx live
<CloseButton />
```

## Sizes

```jsx live
<Box toCenterY spaceX3>
  <CloseButton size="sm" />
  <CloseButton size="md" />
  <CloseButton size="lg" />
  <CloseButton size={30} />
  <CloseButton size={18} />
</Box>
```

## Rounded

```jsx live
<Box toCenterY spaceX3>
  <CloseButton size="sm" roundedFull />
  <CloseButton size="md" roundedFull />
  <CloseButton size="lg" roundedNone />
</Box>
```

## Customize background

```jsx live
<Box toCenterY spaceX3>
  <CloseButton size="sm" bgBlack--T96 bgBlack--T90--hover roundedFull />
  <CloseButton size="md" bgGreen100--T20 />
  <CloseButton size="lg" bgBlack--T96 roundedFull />
</Box>
```

## Customize color

```jsx live
<Box toCenterY spaceX3>
  <CloseButton size="sm" red600 />
  <CloseButton size="md" green600 />
  <CloseButton size="lg" purple600 />
</Box>
```
