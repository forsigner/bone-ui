# Card

## import

```jsx
import { Card } from '@bone-ui/card'
```

## Basic

```jsx live
<Card>A basic card.</Card>
```

## Borderless

```jsx live
<Box bgGray100 p4>
  <Card borderNone>A borderless card.</Card>
</Box>
```

## Shadow

```jsx live
<Card shadow borderNone>
  A card with shadow.
</Card>
```

## Hoverable

```jsx live
<Card shadowMD--hover>A hoverable card.</Card>
```

## With title

```jsx live
<Card spaceY4 w-400>
  <Box textLG fontBold>
    Bone UI
  </Box>
  <Box>
    Lorem ipsum dolor sit amet, consectetur adipisicing elit. Assumenda, quia temporibus quis sed
    voluptatum quae eum fugit earum.

</Card>
```
